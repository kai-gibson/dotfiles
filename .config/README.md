# Kai's Dotfiles for Arch Linux

## To Install:
`git clone https://github.com/kai-gibson/dotfiles.git ~/`
## Includes configs for:
    - Fish shell
    - Kitty terminal emulator
    - Lf file manager
    - Neovim (as an IDE)
    - Rofi
    - Tmux

## And Custom builds of:
    - Dwm
    - Dwm blocks
